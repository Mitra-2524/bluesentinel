#  BlueSentinel

BlueSentinel is an AI-powered system to detect illegal fishing activities using image analysis and monitoring.

##  Features
- Upload images for detection
- FastAPI backend
- React frontend
- Scalable for AI integration

##  Project Structure
backend/ → API services  
frontend/ → User interface  

##  Tech Stack
- FastAPI
- React (Vite)
- Python

## How to Run

### Backend
cd backend  
uvicorn main:app --reload  

### Frontend
cd frontend  
npm install  
npm run dev

- AI fish detection (CNN)
